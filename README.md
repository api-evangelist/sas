# SAS Institute (sas)

SAS Institute is a global leader in analytics, AI, and data management. SAS Viya, its cloud-native platform, exposes a comprehensive catalog of REST APIs spanning analytics, data management, model management, decisioning, business rules, reporting, identities, and platform administration.

**URL:** [https://www.sas.com/](https://www.sas.com/)

**Developer Portal:** [https://developer.sas.com/](https://developer.sas.com/)
